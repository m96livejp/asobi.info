LibreTranslate ローカル翻訳サーバー
====================================

■ 必要環境
  - Python 3.10以上
  - 約1.5GB のディスク容量（言語モデル含む）
  - 約1.5〜2GB のメモリ

■ セットアップ（初回のみ）
  setup.bat をダブルクリック
  → 仮想環境を作成して LibreTranslate をインストールします

■ 起動
  start.bat をダブルクリック
  → http://127.0.0.1:5000 で翻訳サーバーが起動します
  → 初回起動時は言語モデルのダウンロードに時間がかかります

■ 停止
  コマンドプロンプトで Ctrl+C

■ フォルダ構成
  このフォルダごと好きな場所に移動可能です。
  setup.bat / start.bat は %~dp0（自身のフォルダ）を基準に動きます。

  libretranslate/
  ├── setup.bat      ... 初回セットアップ
  ├── start.bat      ... 起動
  ├── README.txt     ... このファイル
  └── venv/          ... （setup後に作成される仮想環境）

■ AIC管理画面での設定
  翻訳モード: 「ローカルのみ」または「無料版→ローカル」
  ローカルエンドポイント: http://127.0.0.1:5000
