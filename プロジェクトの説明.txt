■ 概要
asobi.info は情報サイトとオリジナルゲームで構成されるWebサイトです。
情報コンテンツで集客し、オリジナルコンテンツでユーザーを増やして収益化を目指します。

■ 共通認証システム
=====================================

  全サブドメイン共有のPHP+SQLiteセッション認証
  DB：/opt/asobi/data/users.sqlite

  主要ファイル：
    shared/assets/php/auth.php          認証モジュール
    shared/assets/php/users_db.php      DB接続
    shared/assets/php/oauth_config.php  OAuth設定
    shared/assets/php/me.php            ログイン状態JSON API（CORS対応）
    shared/assets/php/log-access.php    アクセスログ記録API（JS→AJAX）
    info/contact.php                    問い合わせフォーム

  使い方（各サブドメインから）：
    require_once '/opt/asobi/shared/assets/php/auth.php';
    asobiRequireLogin();   // 未ログインでリダイレクト
    asobiRequireAdmin();   // 管理者以外に403

  セッションロック対策：
    me.php・log-access.phpはセッション読み取り直後にsession_write_close()を呼び
    PHPセッションファイルのロックを早期解放する（並列リクエストの遅延防止）

■ Webフォント
=====================================

  使用フォント：Migu フォント（IPA Font License v1.0）
    Migu 1P  プロポーショナル
    Migu 1C  英字改善・全角かなもプロポーショナル（デフォルト）
    Migu 1M  等幅（数字の桁ずれなし）
    Migu 2M  等幅・濁点控えめ

  ライセンス表記：
    © 2002-2019 M+ FONTS PROJECT / © 2003-2011 IPA
    https://asobi.info/licenses.html（ライセンスページ）

  適用ルール（重要・毎回守ること）：
    - body にはシステムフォントを使用
    - main にのみカスタムフォントを適用（タイトル含む）
    - font-display: swap 必須
    - 形式：WOFF2（優先）＋ TTF（フォールバック）
    - 管理画面（/admin/font.php）でフォント種類を切り替え可能

  良い例：
    body { font-family: system-ui, -apple-system, sans-serif; }
    main { font-family: 'Migu 1C', system-ui, sans-serif; }

  悪い例（やってはいけない）：
    body { font-family: 'Migu 1C', sans-serif; }

■ 運用ルール
=====================================

  - デプロイはdeploy.shを使用（旧パス /home/m96 が残っていると自動中止）
  - 動作確認後にGitコミット＆プッシュを徹底
  - Webフォント：font-display:swap必須、mainのみ適用、WOFF2+TTF両用意
  - プレビューツール：最小限の使用（ウィンドウが増えるため）
  - 新規ページ追加時はlicenses.htmlへの著作権・ライセンス追記を忘れずに
  - PHPでセッションを読み取るだけのAPIはsession_write_close()で早期解放すること
  - require_once のパスは /opt/asobi/shared/assets/php/ を使用すること
